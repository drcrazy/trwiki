Recommended Mods‚Äã

Some mods can improve your experience using Tech Reborn. None are strictly required, as you can play all of Tech Reborn using what's provided within the mod and this wiki.

REI‚Äã

Roughly Enough Items. The ultimate companion to any mod that adds blocks and items, it allows you to see recipes and uses for everything in-game. If you're playing on a server, then REI must also be installed on the server.

RER‚Äã

Roughly Enough Resources(RER) is a REI add-on that shows the Y-levels that ores generate.

WTHIT‚Äã

WTHIT adds an in-game tooltip with information about the block (or fluid) the player is looking at.

Megane‚Äã

Megane:Fabric is a WTHIT add-on that includes Tech Reborn compatible features such as energy, fluid volumes, and inventory contents.

Browse wiki in-game‚Äã

info

1.12 Only

This wiki can be viewed in-game using Charset Tablet, a mod created by Asiekierka that allows you to search and read a wide range of mod wikis in-game. Support for Tech Reborn is built in!

Both Tech Reborn and Charset Tablet need to be installed for it to work correctly.
